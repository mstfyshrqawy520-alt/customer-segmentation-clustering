<h1 align="center">🧩 Customer Segmentation using Clustering</h1>
<h3 align="center">Unsupervised Machine Learning Project</h3>

<hr/>

<h2>🚀 Project Overview</h2>
<p>
This project demonstrates customer segmentation using unsupervised 
machine learning techniques applied to mall customer data.
</p>

<p>
Clustering algorithms were used to group customers based on 
spending behavior and demographic features to identify meaningful patterns.
</p>

<hr/>

<h2>📊 Dataset</h2>
<p>
The dataset contains customer information such as:
</p>

<ul>
<li>Customer ID</li>
<li>Gender</li>
<li>Age</li>
<li>Annual Income</li>
<li>Spending Score</li>
</ul>

<p>
Dataset file: <b>Mall_Customers.csv</b>
</p>

<hr/>

<h2>🧠 Clustering Techniques Implemented</h2>

<ul>
<li>K-Means Clustering</li>
<li>Elbow Method (Optimal K selection)</li>
<li>Silhouette Score Evaluation</li>
<li>Hierarchical Clustering (optional)</li>
</ul>

<hr/>

<h2>📈 Workflow</h2>

<ul>
<li>Data Cleaning</li>
<li>Feature Selection</li>
<li>Feature Scaling</li>
<li>Optimal Cluster Selection</li>
<li>Cluster Visualization</li>
<li>Customer Group Interpretation</li>
</ul>

<hr/>

<h2>📊 Results</h2>
<p>
The clustering process successfully segmented customers into distinct groups 
based on income and spending patterns, enabling better marketing strategy decisions.
</p>
